# Your Hero Academia Mod Documentation

This site documents the core gameplay systems, abilities, and quirks in the Your Hero Academia Minecraft mod for NeoForge 1.21.11.

Use this space as the single source of truth for balancing, implementation behavior, and player-facing guidance.

## What is documented here

- **Core Systems**: stamina, progression, and related mechanics.
- **Abilities**: how each ability works and how players use it.
- **Quirks**: each quirk, unlock flow, strengths, and limitations.

## Suggested workflow for writing docs

When you are ready to add real content:

1. Add or edit pages under `docs/core-systems`, `docs/abilities`, or `docs/quirks`.
2. Keep each mechanic or feature on its own page.
3. Use the provided templates to keep entries consistent.
4. Add links between related systems, abilities, and quirks.

## Start here

- Read `getting-started/how-to-use-this-docs`.
- Fill in `core-systems/overview`.
- Add your first real ability entry from `abilities/ability-template`.
- Add your first real quirk entry from `quirks/quirk-template`.

As your mod grows, this structure will keep the project easy to maintain for both development and player onboarding.
