# Your Hero Academia Mod Documentation

This site documents the core gameplay systems, abilities, and quirks in the Your Hero Academia Minecraft mod for NeoForge 1.21.11.

Use this space as the single source of truth for balancing, implementation behavior, and player-facing guidance.

## What is documented here

- **Core Systems**: stamina, progression, and related mechanics.
- **Abilities**: how each ability works and how players use it.
- **Quirks**: each quirk, unlock flow, strengths, and limitations.
